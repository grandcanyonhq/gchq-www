# Grand Canyon HQ Website

It's a static website built with [Jekyll](https://jekyllrb.com). It's nothing fancy, just boring old tech that works.


## Development

```
# Install Ruby, check .ruby-version
bundle install
yarn install
make run
open http://localhost:4001
```

## Deploy

This website is automatically deployed with [Vercel](https://vercel.com/grandcanyonhq/gchq-www) when changes are pushed to GitHub.
